# 第2回
## 電卓(ex02/calc.py)
### 追加機能
- 四則演算：＋以外の四則演算をする
- クリア：entryに入力されている数字、数式の一文字を末尾からdeleteする
- オールクリア：entryに入力されている数字、数式の文字列全体をdeleteする
- パーセント：entryに入力されている数字をパーセント表記にする(100で割る)
- 小数点：小数点を入力可能にし、小数点を用いた計算を可能にする
- ファクトリアル：entryに入力されている数字に対して階乗計算を行う
